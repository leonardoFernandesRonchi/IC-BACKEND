# 📌 IC-BACKEND  
![Status](https://img.shields.io/badge/status-em%20desenvolvimento-yellow)

## 📖 Descrição
Este repositório contém a aplicação **backend** desenvolvida em **Node.js**, responsável por consumir uma API de visão computacional voltada à identificação de leveduras.

Ele atua como intermediário entre o frontend e a API, processando os dados recebidos e disponibilizando-os para a interface.

## ✨ Principais Características
- 🧱 Arquitetura organizada em camadas:
  - Controllers  
  - Services  
  - Models  
  - Routes  
  - Middlewares  
- 🔗 Consumo de API externa de visão computacional
- ⚙️ Processamento e tratamento dos dados
- 🔐 Uso de variáveis de ambiente para segurança
- 🗄️ Suporte a banco de dados relacional com migrations

## 🛠️ Tecnologias Utilizadas
- Node.js  
- Express.js  
- Integração com API de visão computacional  
- Banco de dados relacional
